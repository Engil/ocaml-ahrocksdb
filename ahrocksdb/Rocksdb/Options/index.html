Module Rocksdb.Options

High-level bindings for RocksDB open options

This module provides a binding to the open options available in Rock's C FFIs. It provides a type config which holds the configuration options for opening a Rocksdb database. Configuration settings are then mapped to the appropriate FFI calls when the database is opened.
